# Personal Finance Manager

Application for tracking personal income and expenses with transaction history management.

## Technologies
- Java
- HTML
- CSS
- JavaScript

## Features
- Credit and debit transaction tracking
- User authentication and authorization
- Transaction history management
- Categorization of financial records
- Simple and user-friendly interface

## Description
Personal Finance Manager is a simple application that allows users to manage their daily financial activities by recording credit and debit transactions and maintaining a history of all financial records.

## Future Improvements
- Monthly expense analytics
- Budget planning features
- Data visualization using charts
- Export financial reports
